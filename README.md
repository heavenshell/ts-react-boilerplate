# ts-react-boilerplate

![build](https://github.com/heavenshell/ts-react-boilerplate/workflows/build/badge.svg)

My boilerplate

## Install

```
yarn
```

## Usage

### Build

```
yarn build
```

### Development

```
yarn start
```

### Storybook

```
yarn storybook
```

### Test

```
yarn test
```

### Lint

```
yarn lint
```
